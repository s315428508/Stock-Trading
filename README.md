# 股票交易助手

一个基于Python和tkinter的Windows桌面应用程序，用于管理自选股票并获取交易建议。

## 功能特性

1. **添加自选股票**：通过输入A股股票代码（如000001、600000）添加股票到自选列表
2. **实时价格更新**：点击"更新"按钮获取所有自选股票的实时价格
3. **智能交易建议**：根据多个技术指标（RSI、MA、MACD、成交量、趋势等）综合判断，给出买入/卖出/持有/观望建议
4. **预测准确性**：显示每条建议的预测准确性，帮助评估建议的可信度

## 快速开始

### 方式一：直接运行（需要Python环境）

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **运行程序**
   ```bash
   python stock_trader.py
   ```

### 方式二：打包成exe（推荐，无需Python）

1. **打包程序**
   - 双击运行 `打包.bat`（Windows用户）
   - 或运行 `python build_exe.py`
   - 或手动执行：`pyinstaller --name=股票交易助手 --onefile --windowed stock_trader.py`

2. **找到可执行文件**
   - 打包完成后，在 `dist` 文件夹中找到 `股票交易助手.exe`
   - 直接双击运行，无需安装Python

3. **分享给朋友**
   - 将 `股票交易助手.exe` 复制给其他用户
   - 他们可以直接运行，无需任何配置
