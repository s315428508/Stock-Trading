# 股票交易助手 - 打包说明

## 方法一：使用打包脚本（推荐）

### 步骤：

1. **安装打包工具**
   ```bash
   pip install pyinstaller
   ```

2. **运行打包脚本**
   ```bash
   python build_exe.py
   ```

3. **找到打包好的程序**
   - 打包完成后，在 `dist` 文件夹中会生成 `股票交易助手.exe`
   - 将这个exe文件复制到任何Windows 11电脑上都可以直接运行

## 方法二：手动打包

### 步骤：

1. **安装PyInstaller**
   ```bash
   pip install pyinstaller
   ```

2. **执行打包命令**
   ```bash
   pyinstaller --name=股票交易助手 --onefile --windowed --clean stock_trader.py
   ```

3. **打包选项说明**
   - `--name=股票交易助手`: 生成的exe文件名
   - `--onefile`: 打包成单个exe文件（便于分发）
   - `--windowed`: 不显示控制台窗口（GUI应用）
   - `--clean`: 清理临时文件

## 打包后的文件

- **dist/股票交易助手.exe**: 这是最终的可执行文件
- 可以直接双击运行，无需安装Python

## 文件大小

由于包含了Python解释器和所有依赖库，exe文件大约100-200MB。

## 使用说明

1. 将 `股票交易助手.exe` 复制到目标电脑
2. 双击运行即可
3. 程序会在同目录下创建 `watchlist.json` 文件保存自选股票

## 注意事项

- 首次运行可能需要等待几秒钟（程序自解压）
- 某些杀毒软件可能会误报，需要添加信任
- 确保目标电脑是Windows 10/11系统
